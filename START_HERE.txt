   │ 3. Crea un nuevo proyecto                              │

╔══════════════════════════════════════════════════════════════╗
║  🆘 AYUDA RÁPIDA                                             ║
╚══════════════════════════════════════════════════════════════╝

   ❌ Error "groq.APIConnectionError"
   → Verifica GROQ_API_KEY en backend/.env

   ❌ Error "supabase.exceptions.APIError"  
   → Verifica SUPABASE_URL y SUPABASE_KEY

   ❌ Error "Module not found"
   → cd backend && source venv/bin/activate && pip install -r requirements.txt
   → cd frontend && npm install

   ❌ CORS Error
   → Verifica que backend esté corriendo en puerto 5000

   📖 Más ayuda en: IMPORTANT_NOTES.md
